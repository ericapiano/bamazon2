# bamazon
In this activity I used node.js and mySQL to create a store front from the command line that resembles Amazon.

## getting started
* Clone repo
* Run `npm install` to install the required dependencies.
* Use "ctrl + c" to exit each mode

## Technology Used
* Node.js
* Inquire NPM Package
* MYSQL NPM Package
